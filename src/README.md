# src/

Default working/output root for this project. Unless you name a different
location, `.bat`/`.cmd` scripts you ask for get written here.

Not `examples/` (curated reference scripts, hand-maintained) and not
`docs/` (documentation, not code). See `AGENTS.md`'s "Working Directory"
and "Structure (No Layered Architecture)" sections for the full rule and
the suggested multi-script layout (`lib/`, `logs/`, etc.) inside here.
